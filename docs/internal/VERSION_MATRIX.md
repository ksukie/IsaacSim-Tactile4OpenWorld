# 版本矩阵

| 版本维度 | 标识 | 含义 |
|---|---|---|
| Isaac Lab 基线 | `2.1.1` | 当前 OpenWorldTactile、UIPC、AgileX 主线 |
| Isaac Lab 基线 | `2.3.2` | OpenWorldTactile/GelSight legacy 集成路线 |
| OpenWorldTactile 实验版本 | V1、V2、V4、V5、V6 | 已存在版本命名脚本的主要阶段 |
| 当前参考入口 | V6.2 | OpenWorldTactileBench 当前默认导航，不代表整个仓库版本 |
| 规划/说明阶段 | V3、V7、V8 | 主要以文档形式存在 |

Isaac Lab 2.1.1/2.3.2 与 OpenWorldTactile V1–V6.2 是两个不同的版本维度。
