# 脚本方法与运行指令

范围：

```bash
experiments/franka/
```

说明：

```text
current/ 下的脚本是当前保留脚本。
stable_refs/、high_risk/、superseded/ 下的脚本在本次归档中统一视为失败/历史案例。
```

## 当前保留脚本

### `current/cop_rgb_sdk.py`

采用方法：

```text
ContactSensor -> COP force_map -> OpenWorldTactile textured RGB -> SDK FXYZ -> Isaac UI
```

实现功能：

```text
当前主线脚本。机械臂抓取 cube 后，实时生成左右夹爪的 OpenWorldTactile RGB 图，并用 OpenWorldTactile SDK 解算左右 FXYZ，在 Isaac UI 中同步显示。
```

运行指令：

```bash
env TERM=xterm ./isaaclab.sh -p experiments/franka/current/cop_rgb_sdk.py --max_steps 0 --show_rgb_maps --force_preview_every 1 --force_preview_scale 3
```

### `current/cop_force_map.py`

采用方法：

```text
ContactSensor -> COP + 最大熵 -> dense force_map -> Isaac UI
```

实现功能：

```text
机械臂抓取 cube 后，基于真实接触力生成左右夹爪 97 x 77 的逐点力图，并实时显示灰度 force map。
```

运行指令：

```bash
env TERM=xterm ./isaaclab.sh -p experiments/franka/current/cop_force_map.py --max_steps 0 --show_force_maps --force_preview_every 1 --force_preview_scale 4
```

### `current/raycast_force_map.py`

采用方法：

```text
ContactSensor 门控 -> raycast hit -> 距离权重 -> force_map
```

实现功能：

```text
只有 ContactSensor 检测到真实接触时，才把 raycast 命中的物体区域转换成左右夹爪 force map，可保存 .npy 和调试灰度图。
```

运行指令：

```bash
env TERM=xterm ./isaaclab.sh -p experiments/franka/current/raycast_force_map.py --max_steps 0 --show_force_maps --force_preview_every 1
```

### `current/contact_points_openworldtactile.py`

采用方法：

```text
ContactSensor contact points -> pressure map -> OpenWorldTactile RGB -> SDK CSV
```

实现功能：

```text
根据 ContactSensor 接触点生成 pressure map，再映射为 OpenWorldTactile RGB，可保存 RGB、pressure 调试图和 SDK 输出 CSV。
```

运行指令：

```bash
env TERM=xterm ./isaaclab.sh -p experiments/franka/current/contact_points_openworldtactile.py --max_steps 0 --save_every 1 --max_saved_frames -1 --save_debug_pressure --openworldtactile_image_size 320 240 --output_dir outputs/franka_lift_object_contact_openworldtactile_v2
```

### `current/object_raycast_openworldtactile.py`

采用方法：

```text
object raycast -> deformation/pressure -> OpenWorldTactile RGB -> SDK
```

实现功能：

```text
不依赖 ContactSensor 接触点，直接用虚拟触觉平面对物体做 raycast，生成物体几何相关的 pressure/RGB 结果。
```

运行指令：

```bash
env TERM=xterm ./isaaclab.sh -p experiments/franka/current/object_raycast_openworldtactile.py --max_steps 800 --save_every 1 --max_saved_frames 80 --save_debug_pressure --openworldtactile_image_size 320 240 --output_dir outputs/franka_lift_object_only_openworldtactile
```

### `current/tactile_camera_openworldtactile.py`

采用方法：

```text
Franka lift + runtime tactile camera/OpenWorldTactile RGB 或 depth difference
```

实现功能：

```text
在抓取任务中挂载 runtime tactile camera，采集左右触觉视角图像，可输出 OpenWorldTactile 风格 RGB 或 depth difference 结果。
```

运行指令：

```bash
env TERM=xterm ./isaaclab.sh -p experiments/franka/current/tactile_camera_openworldtactile.py --enable_cameras --max_steps 800 --save_every 5 --max_saved_frames 80 --output_dir outputs/franka_lift_tactile_rgb
```

## 失败案例脚本

### `stable_refs/franka_panda_gripper_basic.py`

采用方法：

```text
最小 Panda + 两指夹爪关节开合
```

失败原因：

```text
只验证夹爪基础控制，没有物体接触、触觉力图、RGB 或 SDK 解算能力。
```

运行指令：

```bash
env TERM=xterm ./isaaclab.sh -p experiments/franka/stable_refs/franka_panda_gripper_basic.py --max-steps 0
```

### `stable_refs/franka_panda_auto_control.py`

采用方法：

```text
Differential IK 控制末端位姿，夹爪单独开合
```

失败原因：

```text
只解决机械臂运动控制，没有稳定接触建模和触觉图生成流程。
```

运行指令：

```bash
env TERM=xterm ./isaaclab.sh -p experiments/franka/stable_refs/franka_panda_auto_control.py --max-steps 0 --stage-steps 220
```

### `stable_refs/franka_panda_contact_sensor_dynamic.py`

采用方法：

```text
Panda 抓动态 cube，左右 finger ContactSensor 输出接触力
```

失败原因：

```text
只能得到每侧手指的接触力结果，无法直接得到夹爪平面上逐点力分布矩阵。
```

运行指令：

```bash
env TERM=xterm ./isaaclab.sh -p experiments/franka/stable_refs/franka_panda_contact_sensor_dynamic.py --max-steps 0 --print-every 20 --touch-threshold 0.1
```

### `stable_refs/franka_panda_grasp_release_loop.py`

采用方法：

```text
IK 抓取/释放循环，左右 finger ContactSensor 打印接触力
```

失败原因：

```text
接触检测可以工作，但输出仍是整体接触力，不能生成稳定的触觉图像。
```

运行指令：

```bash
env TERM=xterm ./isaaclab.sh -p experiments/franka/stable_refs/franka_panda_grasp_release_loop.py --max-steps 0 --print-every 25 --touch-threshold 0.1
```

### `stable_refs/franka_panda_grasp_force_rgb.py`

采用方法：

```text
ContactSensor 三轴力 -> force RGB / OpenWorldTactile RGB -> 可选 SDK 反解
```

失败原因：

```text
把整体三轴力直接映射到 RGB，缺少真实的二维力分布，画面效果不稳定也不够真实。
```

运行指令：

```bash
env TERM=xterm ./isaaclab.sh -p experiments/franka/stable_refs/franka_panda_grasp_force_rgb.py --max-steps 0 --rgb-mode openworldtactile --print-every 25 --touch-threshold 0.1
```

### `stable_refs/franka_panda_tactile_pad_depth.py`

采用方法：

```text
runtime tactile depth camera -> baseline/current depth -> depth difference PNG
```

失败原因：

```text
深度相机差分能得到视觉变化，但和 ContactSensor 的真实接触力没有稳定绑定。
```

运行指令：

```bash
env TERM=xterm ./isaaclab.sh -p experiments/franka/stable_refs/franka_panda_tactile_pad_depth.py --enable_cameras --max-steps 0 --print-every 50 --record-dir outputs/franka_panda_tactile_pad_depth
```

### `high_risk/franka_panda_gelsight_mount_minimal.py`

采用方法：

```text
把 GelSight pads 挂到 Panda fingers，采集 tactile camera/contact 信息
```

失败原因：

```text
直接挂载 GelSight 结构风险较高，传感器姿态、碰撞和相机视野不够稳定，不适合作为当前主线。
```

运行指令：

```bash
env TERM=xterm ./isaaclab.sh -p experiments/franka/high_risk/franka_panda_gelsight_mount_minimal.py --enable_cameras --max-steps 0 --print-every 50 --settle-steps 80
```
