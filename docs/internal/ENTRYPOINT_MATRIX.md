## 结论

- 共登记 153 个 Python 脚本：2.1.1 为 132 个，2.3.2 为 21 个。

- 2.1.1 的当前说明默认参考入口是 `OpenWorldTactile_v6_2_grasp.py`；V1–V6.1b 和其他分支继续完整保留。
